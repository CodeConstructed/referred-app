## Compiling 
How to compile and generate .xcworkspace file

 `git clone https://github.com/uuttff8/Lemmy-iOS.git`   
 
 `cd Lemmy-iOS`
   
 Run [CocoaPods](https://cocoapods.org/) Install
 
 `pod install`
 
 `open Lemmy-iOS.xcworkspace/`


And you are done!
